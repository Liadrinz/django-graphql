# Django 入门

##### 1. Windows下环境搭建
- 安装 python 3.6.5
https://www.python.org/downloads/release/python-365/
- 安装 pip

- 使用 pip 安装 django
```shell
pip install django
```

##### 2. Ubuntu下环境搭建
- 安装 python3 (系统默认是python2)
```shell
sudo apt-get install python3
```
- 安装 pip3 (pip对应python2, pip3对应python3)
```shell
sudo apt-get install pip3
```
- 使用 pip3 安装 django
```shell
pip3 install django
```
